# Security Policy

## Supported Versions

Currently, only the latest generic script directly from `main` is supported with security patches.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within `disk-health-cli` (e.g., unintended privilege escalation with `sudo`, arbitrary command execution via maliciously crafted S.M.A.R.T outputs), please do NOT open a public issue.

Submit a report privately directly to the maintainers of the `blackstart-labs` organization or email the repository owner. We will ensure the issue is acknowledged and patched via the built-in auto-update mechanism as quickly as possible.
